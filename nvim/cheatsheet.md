# Neovim Cheatsheet (LazyVim)

## Completion (Insert Mode)

| Key | Action |
|-----|--------|
| `<C-n>` | Next completion item |
| `<C-p>` | Previous completion item |
| `<CR>` | Accept completion |
| `<C-e>` | Close completion menu |
| `<C-Space>` | Trigger completion |

## Leader Key = `<Space>`

## File/Find (Telescope)

| Key | Action |
|-----|--------|
| `<leader>ff` | Find files |
| `<leader>fg` | Live grep |
| `<leader>fb` | Buffers |
| `<leader>fr` | Recent files |
| `<leader><leader>` | Find files (root) |
| `<leader>/` | Grep (root dir) |

## LSP

| Key | Action |
|-----|--------|
| `gd` | Go to definition |
| `gr` | Go to references |
| `gI` | Go to implementation |
| `gy` | Go to type definition |
| `K` | Hover docs |
| `gK` | Signature help |
| `<leader>ca` | Code actions |
| `<leader>cr` | Rename symbol |
| `<leader>cd` | Line diagnostics |
| `]d` / `[d` | Next/prev diagnostic |

## Buffers & Windows

| Key | Action |
|-----|--------|
| `<S-h>` / `<S-l>` | Prev/next buffer |
| `<leader>bd` | Delete buffer |
| `<leader>bo` | Delete other buffers |
| `<C-h/j/k/l>` | Navigate windows |
| `<leader>-` | Split below |
| `<leader>\|` | Split right |

## Explorer & UI

| Key | Action |
|-----|--------|
| `<leader>e` | File explorer |
| `<leader>E` | File explorer (cwd) |
| `<leader>gg` | Lazygit |
| `<leader>xx` | Trouble diagnostics |
| `<leader>xX` | Buffer diagnostics |

## Obsidian

| Key | Action |
|-----|--------|
| `<leader>on` | New note |
| `<leader>od` | Daily note |
| `<leader>os` | Search notes |
| `<leader>ob` | Backlinks |

## Editing

| Key | Action |
|-----|--------|
| `gcc` | Toggle line comment |
| `gc` | Toggle comment (visual) |
| `]e` / `[e` | Move line down/up |
| `<A-j>` / `<A-k>` | Move line down/up |

## Navigation

| Key | Action |
|-----|--------|
| `gg` / `G` | Start/end of file |
| `{` / `}` | Jump paragraphs |
| `%` | Matching bracket |
| `<C-o>` / `<C-i>` | Jump back/forward |
| `zz` | Center cursor |

## Search & Replace

| Key | Action |
|-----|--------|
| `/` / `?` | Search forward/backward |
| `n` / `N` | Next/prev match |
| `*` / `#` | Search word under cursor |
| `<leader>sr` | Replace in files |

## General

| Key | Action |
|-----|--------|
| `<leader>` | Which-key (shows all bindings) |
| `<Esc>` | Clear search highlight |
| `<leader>qq` | Quit all |
| `<leader>fn` | New file |
| `<leader>xl` | Location list |
| `<leader>xq` | Quickfix list |

---

**Tip:** Press `<Space>` and wait - Which-Key will show you all available keybindings organized by category.
